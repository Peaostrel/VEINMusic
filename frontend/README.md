# VEIN Music Frontend

Веб-интерфейс системы музыкальной статистики VEIN Music.

## 🛠 Технологии
- **Next.js 14+** (App Router)
- **React 19**
- **Tailwind CSS 4**
- **Framer Motion** (Анимации)
- **Lucide React** (Иконки)
- **Recharts** (Графики статистики)

## 🚀 Запуск
Для запуска отдельно от бэкенда:
```bash
npm install
npm run dev
```

Откройте [http://localhost:3000](http://localhost:3000).

## 📁 Структура
- `app/` — Страницы и роутинг.
- `components/` — UI компоненты (профиль, лидерборд, графики).
- `public/` — Статические ресурсы и логотипы.
- `types/` — TypeScript интерфейсы.
