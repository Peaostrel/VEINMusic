**VEIN Music Tracker** — это огромный проект, созданный для настоящих меломанов. Мы объединяем ваши прослушивания из *различных стриминговых сервисов* в едином, футуристичном и красивом профиле пользователя. Слушайте музыку, получайте опыт (XP), открывайте достижения и соревнуйтесь в глобальных Рейтингах!

---

## ⚡ Ключевые возможности

🚀 **Универсальный Скроблинг (VEIN Extension)**  
Независимо от того, где вы слушаете музыку, наше браузерное расширение перехватывает треки и отправляет их в вашу общую статистику. Никаких закрытых API или Premium-подписок — всё работает напрямую из вашего браузера (Яндекс Музыка, Spotify, VK Видео/Музыка, Apple Music, SoundCloud).

🎮 **Игровая система прокачки (Геймификация)**  
- Рассчитывает время прослушивания (считает скроблинг успешным, если прослушано >85% трека).
- За каждый трек начисляется **+10 XP** и **1 V-Coin** (виртуальная валюта проекта).
- Повышайте ранги (от «Туриста» до «Легенды») и открывайте уникальные достижения (например: прослушать 100 треков одного жанра).

🎨 **Премиальный Интерфейс (Стеклянный UI)**  
Все профили построены с использованием современных эффектов стекломорфизма (Glassmorphism), плавающих блюр-эффектов, сглаженных анимаций и динамических цветов, подстраивающихся под обложки альбомов в режиме реального времени.

🌍 **Личная Карта Пользователя**  
Настройте свой профиль: установите страну и город проживания (с автоматическим подтягиванием флага), любимые жанры и даже аудиотехнику (наушники, ЦАПы, колонки).

💬 **Discord Rich Presence (RPC)**  
Нативный Python-скрипт автоматически транслирует то, что вы слушаете в данный момент, прямо в ваш статус Discord, показывая ваш текущий Уровень и Ранг в системе VEIN.

## 🛠 Технологический стек

- **Backend:** `FastAPI` (Python), `SQLAlchemy`, `SQLite` (с возможностью миграции на PostgreSQL/MySQL).
- **Frontend:** `Next.js` (React), `Tailwind CSS`, `Framer Motion` (для анимаций), `Lucide React` (иконки).
- **Extension:** Нативный JavaScript (Chrome Extensions Manifest V3), обход сложных политик CORS и Content-Security-Policy (например, у Яндекса).
- **Integrations:** `PyPresence` (для Discord RPC), `Nominatim API` (для гео-данных стран).

## 📁 Структура проекта

* `frontend/` — Клиентская часть проекта на Next.js.
* `music-extension/` — Исходный код расширения для перехвата аудиоконтекста на сайтах.
* `main.py` — Сердцевина проекта: API Gateway, логика уровней, подсчет времени и регистрация.
* `discord_rpc.py` — Автономный модуль для трансляции музыки в статус Discord.
* `start.bat` — Единый лончер. Автоматически инициирует backend и frontend в изолированных окнах и открывает браузер.

## 🚀 Как запустить на своем ПК?

1. **Клонируйте репозиторий:**
   ```bash
   git clone https://github.com/Peaostrel/VEINMusic.git
   cd VEINMusic
   ```

2. **Backend (Python):**
   - Установите зависимости: `pip install fastapi uvicorn sqlalchemy pypresence colorthief`.
   - Если сервер нужен отдельно: `python -m uvicorn main:app --reload`.

3. **Frontend (Node.js):**
   - Зайдите в папку: `cd frontend`
   - Установите пакеты: `npm install`
   - Запустите сервер: `npm run dev`

4. **Автоматический запуск (Windows):**
   Просто запустите файл `start.bat` в корне проекта (он всё сделает сам).

5. **Установка расширения (VEIN Extension):**
   - Откройте браузер на Chromium (Chrome, Edge, Yandex, Brave, Arc).
   - Перейдите по адресу `edge://extensions/` или `chrome://extensions/`.
   - Включите **Режим разработчика**.
   - Нажмите "Загрузить распакованное расширение" (Load unpacked) и выберите папку `music-extension`.
   - На сайте VEIN (в разделе "Настройки") нажмите кнопку "Подключить VEIN Web Extension", чтобы синхронизировать ваш профиль с расширением.
